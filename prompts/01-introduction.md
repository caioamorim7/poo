* Liste versões importantes do Java e identifique as principais mudanças ou recursos adicionados em cada uma.

* Explique como os princípios da Programação Orientada a Objetos (POO) estão incorporados em Java.
* Crie um exemplo simples de código em Java que demonstre a aplicação dos quatro pilares da POO (Encapsulamento, Herança, Polimorfismo e Abstração). Explique cada parte do código.
* Desenvolva uma pequena hierarquia de classes em Java que represente animais, incluindo atributos e métodos.
* Simule como a aplicação da POO em Java pode facilitar a manutenção de um sistema bancário.
