## 4. Tópicos Essenciais a Dominar

**Instrução:** Esteja preparado para responder com profundidade sobre os seguintes tópicos de Programação Orientada a Objetos em Java:

*   **Fundamentos de POO:**
    *   Classes e Objetos
    *   Encapsulamento, Getters e Setters
    *   Herança e Hierarquia de Classes
    *   Polimorfismo (Sobrecarga e Sobrescrita)
    *   Abstração (Classes e Métodos Abstratos)
    *   Interfaces e Implementações
    *   Modificadores de Acesso (public, private, protected, default)

*   **Conceitos Avançados de POO em Java:**
    *   Classes Internas e Aninhadas
    *   Classes Anônimas
    *   Genéricos (Generics)
    *   Enumerações (Enum)
    *   Anotações (Annotations)
    *   Reflexão (Reflection)
    *   Expressões Lambda e Interfaces Funcionais

*   **Padrões de Projeto em Java:**
    *   Singleton, Factory, Builder, Observer, Strategy, etc.
    *   Implementações práticas dos padrões

*   **Coleções e Estruturas de Dados em Java:**
    *   ArrayList, LinkedList, HashMap, TreeMap, etc.
    *   Iteração e manipulação de coleções

*   **Tratamento de Exceções:**
    *   Try-catch-finally
    *   Exceções verificadas vs. não verificadas
    *   Criação de exceções personalizadas

*   **Concorrência e Multithreading:**
    *   Threads em Java
    *   Sincronização e locks
    *   Executors e ThreadPools

*   **Entrada e Saída (I/O):**
    *   Streams e Readers/Writers
    *   Serialização de objetos

*   **Java 8+ Features:**
    *   Streams API
    *   Optional
    *   Nova API de Data/Hora
    *   Default methods em interfaces

*   **Testes em Java:**
    *   JUnit
    *   Mocking
    *   TDD em Java
