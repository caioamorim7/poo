Problema: Contar Subvetores com Limites Fixos

Dado um vetor de inteiros, `nums`, e dois inteiros `minK` e `maxK`, retorne o número de subvetores com limites fixos.

Um subvetor em `nums` é chamado de subvetor com limites fixos se satisfizer as seguintes condições:

O menor valor no subvetor é igual a `minK`.

O maior valor no subvetor é igual a `maxK`.

Nota: Um subvetor é uma sequência contígua de elementos dentro de um vetor.

Restrições:

2 ≤ 2 ≤ `nums.length` ≤ 10³ ≤ 10³

1 ≤ 1 ≤ `nums[i], minK, maxK` ≤ 10³ ≤ 10³
